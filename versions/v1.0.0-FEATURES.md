# ThreadJS v1.0.0 - Feature Summary

## Overview
ThreadJS v1.0.0 is a comprehensive Minecraft modding framework with visual designer and extensive API.

## New Features (All 20 Categories)

### 1. Block Interaction Events ✓
- `onBlockInteract` - Right-click block events
- `onBlockClick` - Left-click block events
- Full event data: player, block ID, coordinates

### 2. Item Events ✓
- `onItemDrop` - Item drop events
- `onItemPickup` - Item pickup events
- `onItemCraft` - Crafting events
- `onItemConsume` - Food/potion consumption

### 3. Player State Events ✓
- `onPlayerRespawn` - Respawn events
- `onPlayerDimensionChange` - Dimension travel
- `onPlayerGamemodeChange` - Gamemode changes
- `onPlayerSleep` - Bed enter/leave

### 4. Inventory Management ✓
- `getInventory` - Get player inventory
- `clearInventory` - Clear inventory
- `hasItem` - Check for item
- `removeItem` - Remove specific items

### 5. Advanced Entity Operations ✓
- `setVelocity` - Launch entities
- `teleportEntity` - Move entities
- `getNearestEntity` - Find closest entity
- `applyEffect` - Potion effects
- `getEntityNBT` - Access NBT data

### 6. Particle & Visual Effects ✓
- `spawnParticle` - Spawn particles at location
- `spawnParticleLine` - Particle line between points
- `spawnParticleCircle` - Circular particle effect
- `spawnParticleTrail` - Attach trail to entity

### 7. Title & Actionbar Messages ✓
- `showTitle` - Show title/subtitle
- `showActionbar` - Show actionbar text
- `showBossbar` - Create custom bossbars

### 8. Weather & Time Control ✓
- `setWeather` - Control weather (clear/rain/thunder)
- `setTime` - Set world time
- `getTime` - Get current time

### 9. Scoreboard Integration ✓
- `createScoreboard` - Create scoreboards
- `setScore` - Set player score
- `getScore` - Get player score
- `incrementScore` - Increment score
- `displayScoreboard` - Show scoreboard

### 10. NBT Data Access ✓
- `getPlayerNBT` - Player NBT data
- `getEntityNBT` - Entity NBT data
- `getBlockNBT` - Block NBT data

### 11. Team Management ✓
- `createTeam` - Create teams
- `addToTeam` - Add player to team
- `removeFromTeam` - Remove from team
- `getPlayerTeam` - Get player's team

### 12. Advanced Scheduling ✓
- `runLater` - Delay execution (returns taskId)
- `runRepeating` - Repeat execution (returns taskId)
- `cancelTask` - Cancel scheduled task
- `runAt` - Run at specific time
- `runDaily` - Daily scheduled tasks

### 13. Permission System ✓
- `hasPermission` - Check permission
- `addPermission` - Grant permission
- `removePermission` - Revoke permission
- `getPlayerPermissions` - List permissions

### 14. Economy/Data Extensions ✓
- `dataExists` - Check if data exists
- `deleteData` - Delete data
- `incrementData` - Increment numeric value
- `decrementData` - Decrement numeric value

### 15. Structure Operations ✓
- `placeStructure` - Place saved structure
- `saveStructure` - Save region as structure

### 16. Custom GUI/Forms ✓
- `showSimpleForm` - Button selection form
- `showModalForm` - Yes/no confirmation
- `showCustomForm` - Multi-input forms

### 17. Ray Tracing ✓
- `raycast` - Raycast to block
- `raycastEntity` - Raycast to entity

### 18. Region/Zone System ✓
- `createZone` - Create region
- `isInZone` - Check if in zone
- `onZoneEnter` - Zone enter event
- `onZoneLeave` - Zone leave event
- `deleteZone` - Remove zone

### 19. Custom Recipes ✓
- `addRecipe` - Add crafting recipe
- `removeRecipe` - Remove recipe

### 20. World Modification ✓
- `createExplosion` - Spawn explosion
- `spawnLightning` - Spawn lightning
- `setSpawn` - Set spawn point

## Installation

1. Place mod files in `run/config/jsmods/`
2. Start server/client with ThreadJS
3. Access designer at `/docs/ThreadJS/designer.html`
4. Create mods visually or via code
